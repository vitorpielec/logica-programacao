# Lógica de Programação 🧠

Este repositório é destinado ao meu estudo e prática de lógica de programação.

Aqui eu documento exercícios e pequenos desafios para desenvolver meus fundamentos como desenvolvedor.

---

## 🎯 Objetivo

Praticar e fortalecer conceitos básicos de programação, como:
- variáveis e tipos de dados
- condições (if / else)
- loops (for / while)
- funções
- resolução de problemas

---

## 📁 Organização

O repositório contém:

- `exercises/` → exercícios isolados para praticar conceitos específicos  
- `mini-projects/` → pequenos projetos que combinam mais de um conceito

---

## 🧪 Exemplos de exercícios

- Par ou ímpar  
- Tabuada  
- Conversor de unidades  
- Validação de senha  
- Jogo de adivinhação  

---

## 📌 Observação

Este repositório não contém projetos completos, mas sim exercícios e práticas constantes voltadas ao aprendizado.

---

## 🚀 Evolução

O conteúdo será atualizado conforme meu progresso na programação, valeu!
