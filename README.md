# Lógica de Programação 🧠

Este repositório é destinado ao meu estudo e prática de lógica de programação.

Aqui eu documento exercícios e pequenos desafios para desenvolver meus fundamentos como desenvolvedor.

---

## 🎯 Objetivo

Praticar e fortalecer conceitos básicos de programação como:
- variáveis e tipos de dados
- condições (if / else)
- loops (for / while)
- funções
- resolução de problemas

---

## 📁 Estrutura

- `exercises/` -> exercícios isolados para prática de conceitos específicos  
- `mini-projects/` -> pequenos projetos que combinam múltiplos conceitos  

---

## 🧪 Exemplos

- Par ou ímpar  
- Tabuada  
- Conversor de unidades  
- Validação de senha  
- Jogo de adivinhação  

---

## 📌 Observação

Este repositório contém exercícios e práticas constantes, não projetos completos.
